# Part 1: Course Foundation

## Module Overview
Welcome to the first module of **Physical AI & Humanoid Robotics**. This section lays the groundwork for understanding how intelligence can be embodied in physical machines.

## Learning Objectives
By the end of this module, you will understand:
*   The difference between **Digital AI** (Disembodied) and **Physical AI** (Embodied).
*   The core components of a robot: **Actuators**, **Sensors**, and **Compute**.
*   The hardware platforms used in this course (Unitree G1, NVIDIA Jetson).
